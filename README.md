# Aisha Collector - Premium Abayas E-Commerce Website

![Aisha Collector Banner](https://via.placeholder.com/1200x300/000000/FFFFFF?text=Aisha+Collector+-+Premium+Abayas)

## 📖 Overview

Aisha Collector is a fully responsive e-commerce website designed for an abaya and Islamic wear business. The website provides a seamless shopping experience with a focus on visual appeal, mobile optimization, and user-friendly navigation. Built using vanilla HTML, CSS, and JavaScript, this project demonstrates modern front-end development practices without relying on heavy frameworks.

## ✨ Key Features

### 🏠 Home Page
- Hero section with brand messaging
- Featured products showcase
- Category navigation
- Newsletter signup
- Brand story section

### 🛍️ Products Page
- Grid layout for product display
- Product cards with images, prices, and discounts
- Quick view functionality
- Category filtering
- Responsive product galleries

### 🛒 Shopping Cart
- **LocalStorage Integration**: Cart persists across browser sessions
- **Dynamic Updates**: Add, remove, and update quantities in real-time
- **Price Calculations**: Automatic subtotal, discount, and total calculations
- **Size Selection**: Choose from multiple sizes (S-XXL)
- **Image Gallery**: Multiple product images with thumbnail navigation
- **Image Zoom**: Click to zoom product images
- **Free Shipping Progress Bar**: Visual indicator for free shipping threshold
- **WhatsApp Order Integration**: Direct order placement via WhatsApp
- **Clear Cart Option**: Empty cart with confirmation dialog

### 📞 Contact Us Page
- Contact form with validation
- Business hours display
- Google Maps integration (optional)
- Multiple contact methods (phone, email, WhatsApp)
- FAQ section

### 👥 About Us Page
- Brand story and mission
- Team/Founder information
- Quality commitment statement
- Customer testimonials
- Manufacturing process highlights

### 📱 Mobile-First Design
- **Bottom Navigation Bar**: Easy one-handed access on mobile
- **Hamburger Menu**: Clean desktop navigation
- **Touch Optimized**: All interactive elements sized for touch
- **Responsive Images**: Images scale appropriately on all devices
- **Smooth Animations**: Slide-in mobile menu with CSS animations

### 🎨 Visual Features
- **Image Zoom Modal**: Full-screen image viewing
- **Auto-Scroll Related Products**: Infinite scrolling product recommendations
- **Discount Badges**: Automatic discount percentage display
- **Hover Effects**: Interactive elements with smooth transitions
- **Consistent Branding**: Unified color scheme and typography

### 🔧 Technical Features
- **Modular JavaScript**: Separated components for easy maintenance
- **CSS Variables**: Easy theme customization
- **LocalStorage API**: Persistent cart data
- **Font Awesome Icons**: Professional icon set
- **No Backend Required**: Pure front-end solution
- **Cross-Browser Compatible**: Works on all modern browsers
